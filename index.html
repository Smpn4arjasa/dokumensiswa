Cari Dokumen dengan NISN

Rapor, sertifikat, dan dokumen lainnya tersimpan di Google Drive.

NISN

Contoh NISN: 0012345678

📁 Struktur Folder Google Drive

1
Buat satu Folder Utama di Google Drive sebagai induk semua data siswa.
2
Di dalam folder utama, buat sub-folder per siswa dengan nama yang mengandung NISN.
Contoh: 0012345678_BudiSantoso atau 0087654321_SitiRahayu
3
Masukkan file PDF (rapor, sertifikat, dll.) ke dalam folder masing-masing siswa.
4
Klik kanan Folder Utama → Share → Anyone with the link → Viewer. Lakukan hal sama untuk tiap sub-folder.

Sedang mencari data...
